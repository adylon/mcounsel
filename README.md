# Mcounsel
Payment processing framework

### Task:
This company created a unified purchase processing software that intergrates with other applications for managing your finances. The data FTPS to a server in a shared drive, but all the data is raw xml. They asked for a tool that converts the data and stores to a database with GUI display. 

### Features:
- Moves raw xml customer purchase data from shared folder, converting and storing data in local database file displayed through GUI portal on server

```bash
git clone https://github.com/adylon/mcounsel.git
cd mcounsel
python mcbe.py
python mc.py
```
